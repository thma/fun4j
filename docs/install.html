fun4j - functional programming for the JVM

Getting you started

  1. Download the fun4j.jar from the download page .
  2. Place the fun4j.jar in your Java Classpath to access the Java API
  3. Call 
    java -jar fun4j.jar
    to start up the fun4j Lisp Interpreter.
  4. Checkout the sourcecode from the svn repository by 
    svn co https://fun4j.svn.sourceforge.net/svnroot/fun4j fun4j
    to work directly on the source.
